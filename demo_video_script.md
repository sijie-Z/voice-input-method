# Demo 视频脚本（约 60 秒）

> 录制时打开一个记事本或 Word 作为输入目标窗口，控制台在旁边可见。

| 时间 | 操作 | 台词 |
|------|------|------|
| 0:00 | 屏幕显示程序启动，控制台打印启动横幅 | "大家好，这是一个离线优先、AI 增强的语音输入法。完全在本地运行，不需要网络。" |
| 0:05 | 打开记事本，光标定位。按住右 Ctrl 说"今天下午三点会议室讨论需求方案" | "按住右 Ctrl 开始说话——今天下午三点会议室讨论需求方案。" |
| 0:12 | 松开右 Ctrl，文本自动上屏到记事本 | "松开后自动识别并上屏。全程离线，音频不离开本机。" |
| 0:18 | 按住右 Ctrl，说"邮件模式" | "说'邮件模式'——" |
| 0:22 | 松开后控制台显示"已切换到邮件模式"。接着说"请各位提前阅读需求文档并于周五前反馈" | "切换到邮件润色模式。再说一段草稿——请各位提前阅读需求文档并于周五前反馈。" |
| 0:33 | 松开上屏，文本被润色为邮件格式 | "AI 自动将口语转为正式邮件格式，称呼和落款一并生成。" |
| 0:38 | 按右 Ctrl 说"换行"，松开 | "说'换行'——直接执行而非上屏，整个编辑过程不需要碰键盘。" |
| 0:42 | 按右 Ctrl 说"全部复制"，松开 | "说'全部复制'——" |
| 0:46 | 切换到浏览器，按右 Ctrl 说"粘贴" | "切换到浏览器，说'粘贴'，Ctrl+V 直接执行。" |
| 0:50 | 按右 Ctrl 说"搜索 七牛云 XEngineer 暑期实训" | "说'搜索 七牛云 XEngineer 暑期实训'——" |
| 0:55 | 松开后自动打开浏览器搜索结果 | "自动打开搜索引擎，零手动操作。" |
| 0:58 | 镜头回到控制台，显示完整日志 | "离线识别、语音指令、AI 润色，三步闭环，谢谢观看。" |
